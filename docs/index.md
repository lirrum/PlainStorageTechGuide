# PLST
Plain Storage Tech Guide(以下简称"PSTL")是一个关于MC储电的系统教材。该项目旨在为储电领域开创一部系统而完备的教科书，从一线研究者的视角剖析主流储电体系，以适合新人的学习路线将琐碎的技术细节串在一起。无论你是想系统学习储电的萌新，还是求固本查漏的学者，亦或是寻进阶拔高的老饕，都能从本书中取得一定收获。

## 读前必看

- 书中部分文本使用了特殊颜色。其中,[红色文本]<red>表示需要注意的重点语句，[蓝色文本]<blue>表示指向了其他资源（左键点击[蓝色文本]<blue>进而跳转至对应资源），[绿色文本]<green>表示专有名词或特殊概念。
- 许多储电概念上午统一说法，同时，为系统讲解储电，本书开创了很多新概念，对于这部分内容，会在文本前用倒三角符号（▽）注明。
- 每章末都配套了练习题，既有理论测验，也有实践操作。笔试题答案已遮盖，当鼠标移动到答案上时去除遮盖，左键一次可去除遮盖状态，再次左键可恢复默认状态。像这样:=~awa~=
- 教材内容包含部分主管简介，请选择性阅读，勇于提出自己的观点。
- 书中涉及的所有装置的投影、实验数据、图片等资源均存放在resoures分支中了。
- 因国内访问Github较慢，静态网站资源已镜像至[Gitee](https://gitee.com/lirrum)中。
